----------------------------------------------------------------
INTRODUCTION
----------------------------------------------------------------
Welcome to "Math Challenge Hub," an interactive web application 
designed to help users master mental arithmetic and rapid 
calculation techniques. 

Instead of simple drills, this app focuses on teaching specific 
mathematical tricks (like Vedic math principles) to help you 
solve complex problems in seconds without a calculator. It 
features a fun, gamified interface with cute characters.

----------------------------------------------------------------
KEY FEATURES
----------------------------------------------------------------

1. Diverse Challenge Types
   Practice a variety of arithmetic problems, including:
   - Two-digit multiplication (e.g., 17 x 18, 23 x 27)
   - Squaring numbers (e.g., 45^2, 38^2)
   - Square roots (e.g., sqrt(841))
   - Reverse calculation puzzles

2. Step-by-Step Learning
   Stuck on a hard problem? The "Hint Step" feature breaks down 
   the mental math logic.
   
   [Example from the app]
   - Question: 43 x 47
   - Hint Logic: Since the tens digit is the same (4) and units add up to 10 (3+7)
   - Step 1: 4 x (4 + 1) x 100 = 2000
   - Step 2: 3 x 7 = 21
   - Result: 2000 + 21 = 2021

3. Gamified Experience
   - Math Buddies: Solve problems alongside cute avatars (like 
     the "Pikachu" and "Bulbasaur" styles shown in the app).
   - Stats Tracking: Keeps track of your Correct Answers, Best 
     Score, Best Time, and Last Try.

4. User-Friendly Design
   - Responsive layout that works on desktop and mobile.
   - QR Code on the homepage for quick mobile access.

----------------------------------------------------------------
HOW TO PLAY
----------------------------------------------------------------
1. Choose a Challenge: Click a colored button on the main dashboard 
   (e.g., "21 x 27" or "Basic Math Challenge").
2. Solve: Enter your answer in the input box.
3. Need Help?: 
   - Click "Hint Step" to see the calculation trick.
   - Click "Show Answer" if you are completely stuck.
4. Submit: Click the Submit button to check your result.

----------------------------------------------------------------
MATH TRICKS COVERED
----------------------------------------------------------------
The application includes logic for several famous mental math 
shortcuts, such as:
- Multiplying numbers close to a base (e.g., close to 10 or 20).
- Squaring numbers ending in 5.
- Multiplying numbers where the first digits are the same and 
  the last digits sum to 10.
